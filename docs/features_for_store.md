# Tackle Assist - ストア掲載用 機能一覧

アプリストア（App Store / Google Play 等）のスクリーンショットや説明文を作成するための機能一覧まとめです。画像作成時のキャッチコピーや見せ方の参考にしてください。

---

## 1. タックル・釣り具の在庫デジタル管理
**キャッチコピー案:** 「もう二度と同じルアーは買わない。所持タックルをスマホで一元管理」
* **機能詳細:**
  * ロッド、リール、ルアー、ライン、ウェアなどカテゴリ別の登録。
  * 詳細なスペック（号数、自重など）や「保管場所（例：車、ガレージ）」の管理。
  * 現場でロストしたり消費した際の「ワンタップ削除（数量減少）」機能。
  * お気に入りタックルを「編成（セット）」として保存する機能。

## 2. 釣行パッキング ＆ 忘れ物防止チェックリスト
**キャッチコピー案:** 「次回の釣行をもっとスマートに。忘れ物ゼロの自動パッキング」
* **機能詳細:**
  * 保存したタックルセットを呼び出し、一瞬で「釣行チェックリスト」を作成。
  * TODOリスト形式で持ち物をチェックし、準備状況をパーセンテージで可視化。
  * **アラート連動**: 出発前日などに「準備リマインド通知」を設定可能。
  * 完了した釣行（パッキング履歴）はホーム画面やカレンダーに大切に記録。

## 3. メンテナンス・アラート（状態管理）
**キャッチコピー案:** 「リールもラインも常に最高の状態で。次回メンテを自動でお知らせ」
* **機能詳細:**
  * ラインの巻き替えやリールオーバーホールなどの予定日を管理。
  * **繰り返し設定**: 「完了」ボタンを押すと、自動的に1ヶ月後や半年後の次回アラートを生成。
  * カレンダー上で、いつ何のメンテを行ったか一目で確認可能。

## 4. AI釣りコンシェルジュ機能
**キャッチコピー案:** 「あなたの在庫からAIが最適な仕掛けを提案。専属フィッシングアドバイザー」
* **機能詳細:**
  * 最新の生成AI（Gemini）を利用し、ターゲット魚種や行く予定の釣り場を入力するだけで、**あなたが今持っている釣り具の中から**最適なセットを自動提案。
  * 足りないアイテムがある場合は代替案やおすすめの購入品をアドバイス。
  * そのまま「パッキングリスト」を自動作成することも可能。

## 5. 買い物リスト ＆ 補充機能
**キャッチコピー案:** 「釣り具屋で迷わない。欲しいもの・消耗品をスマートにメモ」
* **機能詳細:**
  * ロストしてしまったルアーや、次に欲しいアイテムを瞬時にメモ。
  * 完了済み（購入済み）リストとの切り替え、一括整理機能付き。

## 6. 釣り専用 単位一括変換ツール
**キャッチコピー案:** 「号数？ポンド？もう迷わない。釣り人専用の単位早見表」
* **機能詳細:**
  * PEラインの号数・ポンド・強度（kg）などをスライダーで瞬時に全自動変換。
  * ナイロン・フロロ・エステルなど複数素材に一括対応。
  * シンカー（重り）の「オンス（oz）」から「グラム（g）」への変換。

### ストア画像の構成案（全5〜6枚推奨）
* **1枚目（メイン）**: アプリの顔となる機能（在庫一覧 あるいは ホームダッシュボード）。
* **2枚目（TODO）**: チェックボックスが並んだパッキング画面（忘れ物防止のアピール）。
* **3枚目（AI）**: AIが回答してくれているオシャレなチャット画面（最新技術のアピール）。
* **4枚目（メンテ）**: メンテ時期のリマインド画面とカレンダー。
* **5枚目（ツール）**: 単位変換ツール（釣り人ならではの細かい悩みを解決しているアピール）。
